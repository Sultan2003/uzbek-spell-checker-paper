# Developing and Evaluating an Uzbek Spell-Checking Framework: Rule-Based and Transformer-Based Approaches

## Project Description

This repository supports the development of a publishable research paper and accompanying experimental framework for Uzbek spell checking. The project investigates both rule-based and transformer-based approaches for detecting and correcting spelling errors in Uzbek, with attention to the linguistic characteristics of the language and the practical requirements of reproducible NLP research.

## Research Question

How can rule-based linguistic methods and transformer-based neural models be developed, evaluated, and compared for robust Uzbek spell checking, and what types of spelling errors remain challenging for each approach?

## Objectives

* Review existing work on spell checking, grammatical error correction, and low-resource NLP, with emphasis on Turkic and morphologically rich languages.
* Define an Uzbek spelling error taxonomy covering general edit-distance errors and Uzbek-specific orthographic phenomena.
* Design a rule-based baseline for Uzbek spelling detection and correction.
* Develop transformer-based models for Uzbek spelling correction using appropriate datasets and evaluation protocols.
* Construct or curate datasets for training, validation, and testing.
* Evaluate models using quantitative metrics and qualitative error analysis.
* Produce a research paper suitable for submission to an AI, NLP, or computational linguistics venue.

## Expected Contributions

* A structured framework for Uzbek spell-checking research.
* A documented taxonomy of Uzbek spelling error types.
* Reproducible experiments comparing rule-based and transformer-based approaches.
* Baseline resources and templates for future Uzbek NLP research.
* A manuscript draft suitable for refinement toward conference, journal, or PhD application purposes.
