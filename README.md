# Developing and Evaluating an Uzbek Spell-Checking Framework: Rule-Based and Transformer-Based Approaches

## Research Question

How do rule-based and transformer-based spell-checking methods compare in correcting spelling errors in Uzbek text?

## Objectives

1. Build an Uzbek spell correction dataset.
2. Implement a rule-based spell checker.
3. Fine-tune transformer models for spell correction.
4. Compare performance using standard NLP metrics.
5. Analyze Uzbek-specific spelling errors.
